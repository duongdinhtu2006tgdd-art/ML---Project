# ML---Project
A linear regression model predicting water quantity using simulated data. The purpose of this project is to learn AI.

## Project description
this project is basically a small experiment I did with linear regression. It's part of a bigger direction I want to pursue, and more importantly, it helped me understand the basic workflow of building a Machine Learning model from scratch. 

The dataset in this project is completely simulated, so it doesn't reflect real-world conditions. I tested the model on 10 unseen samples and got an errorr of less thann 0.1 liters, which looks pretty good at first. But honestly, since the data isn't real and the system is still very simplified, this result doesn't mean much yet. 

Still, I built this project seriously on my own, from data processing to training and evaluation. For me, this is jusst the starting point of my jourrney into AI/ML and eventually Edge AI.
